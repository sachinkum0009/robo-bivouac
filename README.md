# robo-bivouac
Decentralized communication for swarm robots with adhoc

![Robo-Bivouac](docs/media/robobivouac.png)

This project utilizes the BATMAN-adv protocol to enable decentralized communication among swarm robots. The system is designed to facilitate efficient and reliable communication in environments where traditional centralized communication methods may not be feasible.

> [!NOTE]
> Signal Aware Exploration (SAE)
> The scenario it targets is a swarm of robots that perform a task together, such as search and rescue, environmental monitoring, or agricultural tasks. The robots need to communicate with each other to coordinate their actions, share information, and make collective decisions.
